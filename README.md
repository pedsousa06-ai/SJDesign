## 💻 Sobre o projeto

O SJDesign é um projeto desenvolvido com foco na criação de uma interface moderna, responsiva e intuitiva, aplicando conceitos fundamentais de desenvolvimento Front-End e boas práticas de organização de código.

O projeto tem como objetivo apresentar uma experiência visual agradável, destacando design, usabilidade e adaptação para diferentes dispositivos.

## 🚀 Como executar o projeto

1 - Baixe ou clone o projeto

2 - Abra a pasta do projeto

3 - Execute o arquivo `index.html` em seu navegador

### Pré-requisitos

Antes de começar, você vai precisar ter instalado em sua máquina:

* Um navegador atualizado (Google Chrome, Microsoft Edge, Firefox, etc.)
